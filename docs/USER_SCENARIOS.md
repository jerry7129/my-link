# 마이링크 (My Link) 사용자 시나리오

## 1. 방문자 (Visitor): 마이링크 페이지 방문 과정

### 1-1. 프로필 열람 및 상호작용
- **접속**: 방문자는 특정 크리에이터나 소유자의 링크 모음을 보기 위해 제공된 고유 URL(`mylink.com/displayName`)에 접속한다.
- **정보 확인**: 방문자는 이 링크 모음의 소유자가 누구인지 확인하기 위해 상단에 위치한 구글 프로필 사진(photoURL), 실제 이름(username), 그리고 한 줄 소개(bio)를 읽는다.
- **프로필 공유 (Share)**: 방문자는 이 유용한 프로필 페이지 자체를 다른 사용자에게 전달하기 위해 관리자가 상단에 배치한 **'링크 복사하기(Share)'** 아이콘을 클릭한다.
- **링크 탐색**: 방문자는 자신이 원하는 콘텐츠(비즈니스 채널, 포트폴리오 등)를 찾기 위해 화면에 세로로 나열된 링크 블록들을 스크롤하여 탐색한다.
- **링크 이동**: 방문자는 해당 목적지 사이트로 이동하기 위해 관심이 가는 특정 링크 버튼(API로 렌더링된 파비콘 아이콘 + 제목)을 클릭한다. 이때 원래의 마이링크 프로필 창을 이탈하지 않기 위해 **새 창(새 탭)**으로 목적지 사이트가 열린다.

---

## 2. 소유자 (Owner): 링크 추가/수정/삭제 관리 과정

### 2-1. 관리 화면 진입 및 조회
- **로그인**: 소유자는 본인의 프로필과 링크 데이터를 관리하기 위해 전용 구글 계정을 통해 소셜 로그인을 진행한다.
- **현황 확인**: 소유자는 방문자가 내 페이지를 어떻게 바라보게 될지 직관적으로 확인하기 위해, 진입 직후 나타나는 '방문자 뷰 형태의 관리 레이아웃' 내에서 내 프로필 정보와 저장된 기존 링크 목록을 조회한다.

### 2-2. 신규 링크 추가 과정
- **추가 진입**: 소유자는 새로운 포트폴리오나 웹사이트를 등록하기 위해 리스트의 **'새 링크 추가'** 텍스트(또는 버튼)를 클릭한다.
- **정보 입력 (인라인 작성)**: 소유자는 별도의 팝업이나 모달 창 없이 화면 내 해당 영역이 텍스트 폼으로 변환되는 **인라인 편집 인터페이스** 안에서, 연결할 전체 URL과 버튼에 표시될 제목을 텍스트로 입력 후 완료 단추를 누른다.
- **결과 확인**: 소유자는 등록이 완료됨과 동시에 시스템의 구글 파비콘 API가 URL을 인식하여 적절한 아이콘을 버튼 좌측에 자동 렌더링하고, 완성된 블록을 목록에 최종 반영하는 것을 즉시 확인한다.

### 2-3. 기존 링크 수정 및 삭제 과정
- **링크 수정 (인라인 편집)**: 소유자는 이전에 등록했던 채널의 주소가 변경되거나 내용 오타를 고치기 위해, 기존 링크 블록 영역의 텍스트를 마우스로 직접 클릭하여 올바른 내용으로 타이핑해 즉시 변경한다.
- **링크 삭제**: 소유자는 더 이상 운영하지 않는 폐쇄 채널을 목록에서 완전히 제거하기 위해 해당 링크 블록 우측 끝에 위치한 **'삭제' (휴지통 아이콘)** 단추를 클릭한다.
- **삭제 컨펌(경고 알림)**: 소유자는 실수로 소중한 데이터를 날리는 것을 방지하기 위해 화면에 팝업되는 시스템 삭제 경고 창에서 **"정말로 삭제하시겠습니까?"**라는 질문을 확인 후 **'동의(확인)'**를 누른다.
